Naive Bayes classifier to predict most likely tag of a given word and its previous words tag.

Training file is included but a custom file can also be used.

How to run:
---java 8 required

compile - javac NaiveBayesTagging.java

run - java NaiveBayesTagging <Training Corpus POS Tagged>
eg - java NaiveBayesTagging POSTaggedTrainingSet.txt

After running, 
press 1 to enter 2 things:
1) Word for which tag is to be found
2) Tag of previous word.

press anything except 1 to terminate.
